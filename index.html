🎨 Next-Gen CSS Motion Blur Effects

A Proposal for Advanced Visual Control in Web Design

✨ Introduction

The motion-blur() property offers directional and dynamic blur effects for next-level animations and UI transitions — beyond the traditional Gaussian blur.

🧪 Why It Matters

  • Simulated movement and speed
  • Enhanced UI transitions
  • Depth-of-field and layered motion
  • Creative trails and visual storytelling

🔧 Syntax

motion-blur: <type>(<amount>, <direction>);
<type>: linear, radial, or progressive
<amount>: Intensity in px (e.g., 12px)
<direction>: to right | to bottom | from center | etc.

🌟 Types & Examples

1️⃣ motion-blur: linear()

Effect: Applies blur consistently in one direction. Useful for horizontal/vertical movement.

.element {
  motion-blur: linear(12px, to right);
}

2️⃣ motion-blur: radial()

Effect: Radiates from a center point. Mimics explosion or focus effect.

.element {
  motion-blur: radial(15px, from center);
}

3️⃣ motion-blur: progressive()

Effect: Trailing blur that increases with movement. Good for scroll-based effects.

.element {
  motion-blur: progressive(10px, to bottom);
}

💡 Pro Tips

  • Combine with transform, transition, and filter for rich animations.
  • Use responsibly to maintain performance.
  • Check CSSWG status for browser support.
